
# Description

[Nudgyt](https://github.com/lordgape/nudgyt) Authentication System using Nest.js and GraphQL.

![Tests](https://github.com/lordgape/nudgyt/actions/workflows/ci.yml/badge.svg)

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev
```

## Test

```bash
# unit tests
$ npm run test
```
